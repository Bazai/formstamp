Tags

Description

Directive: w-tags

Tags allows to select multiple values from a predefined list or to enter custom values. Only primitive types are supported as predefined values. Supports several angular directives: ng-model, ng-disabled, ng-required. Available options are items and limit. Attributes: items and limit.

Example

<form role="form" ng-controller='TagsCtrl'>
  <div class="form-group">
    <label>Choose your element</label>
    <div w-tags="true"
         items="items"
         limit='20'
         ng-model="elements">
    </div>
    Your elements are:
    <ul>
      <li ng-repeat='element in elements'>{{element}}</li>
    </ul>
  </div>
</form>
angular.module('angular-w-demo')
.controller('TagsCtrl', function($scope) {
  $scope.items = [ 'earth', 'water', 'fire', 'wind', 'void' ];
});

Select WIP

Description

Directive: w-chz

This input provides default select-like behavior. It allows to select only one item from predefined list or none of them. Supports several angular directives: ng-model, ng-disabled, ng-required. In ng-model it holds full object, not only value.

Demo

{{item.label || 'none'}}

Miyamoto was awarded by imperator for {{coins.id}} ({{coins.label}}) coins.

Option items

Required: true

Option used for setting items list on w-chz. Value in it should be a list of objects with two fields: id and label. They can be overriden by key-attr and value-attr options respiectively.

Example

<form role='form' ng-controller='ChzItemsCtrl'>
  <div class='form-group'>
    <label>Select weapon:</label>
    <div w-chz items='weapons'></div>
  </div>
  <p>
    Miyamoto was awarded by imperator for 
    <b>{{coins.id}} ({{coins.label}})</b> coins.
  </p>
</form>
angular.module('angular-w-demo')
  .controller('ChzItemsCtrl', function($scope) {
    $scope.weapons = [
      {id: 'katana', label: 'Long sword Katana'},
      {id: 'yuml', label: 'Longbow Yuml'},
      {id: 'yari', label: 'Spear Yari'}
    ];
});

Demo

Option limit

Required: false

If it is set - limits number of items will be used.

Example

<form role='form' ng-controller='ChzLimitsCtrl'>
  <div class='form-group'>
    <label>Select number of weapons:</label>
    <div w-chz items='numbers' ng-model='numberOfWeapons'></div>
  </div>
  <div class='form-group'>
    <label>Select weapon:</label>
    <div w-chz items='weapons' limit='numberOfWeapons.id'></div>
  </div>
</form>

angular.module('angular-w-demo')
.controller('ChzLimitsCtrl', function($scope) {
  $scope.numbers = [
    {id: 1, label: 1},
    {id: 2, label: 2},
    {id: 3, label: 3}
  ];
  $scope.weapons = [
    {id: 'katana', label: 'Long sword Katana'},
    {id: 'yuml', label: 'Longbow Yuml'},
    {id: 'yari', label: 'Spear Yari'}
  ];
});

Demo

Options key-attr and value-attr

Required: false

Overrides default names of fields for id and label of item. Default values: key-attr = 'id', value-attr = 'label'

Example

<form role='form' ng-controller='ChzKeyValuesCtrl'>
  <div class='form-group'>
    <label>Select city of Japan:</label>
    <div w-chz items='cities' key-attr='code' value-attr='name'></div>
  </div>
</form>
angular.module('angular-w-demo')
.controller('ChzKeyValuesCtrl', function($scope) {
  $scope.cities = [
    {code: 'JP SPK', name: 'Sapporo'},
    {code: 'JP SDJ', name: 'Sendai'},
    {code: 'JP TYO', name: 'Tokyo'},
    {code: 'JP NGO', name: 'Nagoya'}
  ];
});

Demo

Templating

Required: false

If some inner html is provided for directive tag, then those html will be used for rendering selected value. Selected item is accessible from templating by item variable. By default item['label'] will be rendered or if value-attr is provided, then item[value-attr] will be used.

Example

<form role='form' ng-controller='ChzTemplatingsCtrl'>
  <div class='form-group'>
    <label>Select city of Japan:</label>
    <div w-chz items='cities'>
      <div ng-show='item'>
        Code: <b>{{item.id}}</b>, Name: <b>{{item.label}}</b>
      </div>
      <div ng-hide='item'>
        No city selected
      </div>
    </div>
  </div>
</form>
angular.module('angular-w-demo')
.controller('ChzTemplatingsCtrl', function($scope) {
  $scope.cities = [
    {id: 'JP SPK', label: 'Sapporo'},
    {id: 'JP SDJ', label: 'Sendai'},
    {id: 'JP TYO', label: 'Tokyo'},
    {id: 'JP NGO', label: 'Nagoya'}
  ];
});

MultiSelect WIP

Description

Directive: w-multi-select

MultiSelect allows to select multiple items from a predefined list of options. Supported angular directives and options are similar to Select.

Example

<form role='form' ng-controller='MultiSelectCtrl'>
  <div class='form-group'>
    <label>Leisure Time:</label>
    <div w-multi-select items='games'>
      'none'
    </div>
  </div>
</form>
angular.module('angular-w-demo')
.controller('MultiSelectCtrl', ['$scope', function($scope) {
  $scope.games = [
    {id: 0, label: 'Dragon Quest'},
    {id: 1, label: 'Final Fantasy'},
    {id: 2, label: 'Shin Megami Tensei'}
  ];
}]);

Combo WIP

Description

Directive: w-combo

Combo provides basic combo behavior. It allows to select a value from a predefined list or to enter custom value. Only primitive types are supported as predefined values. Supports several angular directives: ng-model, ng-disabled, ng-required. Available options are items and limit.

Example

<form role="form" ng-controller='ComboCtrl'>
  <div class="form-group">
    <label>Choose your element</label>
    <div w-combo="true" items="items" ng-model="value"></div>
  </div>
</form>
angular.module('angular-w-demo')
  .controller('ComboCtrl', ['$scope', function($scope) {
    $scope.items = [
      'earth',
      'water',
      'fire',
      'wind',
      'void'
    ];
  }]);

Demo

Tags WIP

Description

Directive: w-tags

Tags allows to select multiple values from a predefined list or to enter custom values. Only primitive types are supported as predefined values. Supports several angular directives: ng-model, ng-disabled, ng-required. Available options are items and limit. Attributes: items and limit.

Example

<form role="form" ng-controller='TagsCtrl'>
  <div class="form-group">
    <label>Choose your element</label>
    <div w-tags="true"
         items="items"
         limit='20'
         ng-model="value"></div>
  </div>
</form>
angular.module('angular-w-demo')
  .controller('TagsCtrl', ['$scope', function($scope) {
    $scope.items = [
      'earth',
      'water',
      'fire',
      'wind',
      'void'
    ];
  }]);

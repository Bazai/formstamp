Select WIP

Description

Directive: w-chz

This input provides default select-like behavior. It allows to select only one item from predefined list or none of them. Supports several angular directives: ng-model, ng-disabled, ng-required. In ng-model it holds full object, not only value.

Demo

{{item.label || 'none'}}

Miyamoto was awarded by imperator for {{coins.id}} ({{coins.label}}) coins.

Option items

Required: true

Option used for setting items list on w-chz. Value in it should be a list of objects with two fields: id and label. They can be overriden by key-attr and value-attr options respiectively.

Example

<form role='form' ng-controller='ChzItemsCtrl'>
  <div class='form-group'>
    <label>Select weapon:</label>
    <div w-chz items='weapons'>
      {{(item || 'none') | json}}
    </div>
  </div>
</form>
angular.module('angular-w-demo')
  .controller('ChzItemsCtrl', function($scope) {
    $scope.weapons = [
      {id: 'katana', label: 'Long sword Katana'},
      {id: 'yuml', label: 'Longbow Yuml'},
      {id: 'yari', label: 'Spear Yari'}
    ];
});

Demo

{{(item || 'none') | json}}

Option limit

Required: false

If it is set - limits number of items will be used.

Example

<form role='form' ng-controller='ChzLimitsCtrl'>
  <div class='form-group'>
    <label>Select number of weapons:</label>
    <div w-chz items='numbers' ng-model='numberOfWeapons'>
      {{item.id || 'none'}}
    </div>
  </div>
  <div class='form-group'>
    <label>Select weapon:</label>
    <div w-chz items='weapons' limit='numberOfWeapons.id'>
      {{item.id || 'none'}}
    </div>
  </div>
</form>

angular.module('angular-w-demo')
.controller('ChzLimitsCtrl', function($scope) {
  $scope.numbers = [
    {id: 1, label: 1},
    {id: 2, label: 2},
    {id: 3, label: 3}
  ];
  $scope.weapons = [
    {id: 'katana', label: 'Long sword Katana'},
    {id: 'yuml', label: 'Longbow Yuml'},
    {id: 'yari', label: 'Spear Yari'}
  ];
});

Options key-attr and value-attr

Required: false

Overrides default names of fields for id and label of item. Default values: key-attr = 'id', value-attr = 'label'

Example

<form role='form' ng-controller='ChzKeyValuesCtrl'>
  <div class='form-group'>
    <label>Select city of Japan:</label>
    <div w-chz items='cities' key-attr='code' value-attr='name'>
      <div ng-show='item'>
        Code: {{item.code}}, Name: {{item.name}}
      </div>
      <div ng-hide='item'>
        No city selected
      </div>
    </div>
  </div>
</form>
angular.module('angular-w-demo')
.controller('ChzKeyValuesCtrl', function($scope) {
  $scope.cities = [
    {code: 'JP SPK', name: 'Sapporo'},
    {code: 'JP SDJ', name: 'Sendai'},
    {code: 'JP TYO', name: 'Tokyo'},
    {code: 'JP NGO', name: 'Nagoya'}
  ];
});

Templating

Required: false

If some inner html is provided for directive tag, then those html will be used for rendering selected value. Selected item is accessible from templating by item variable. By default item['label'] will be rendered or if value-attr is provided, then item[value-attr] will be used.

Example

<form role='form' ng-controller='ChzTemplatingsCtrl'>
  <div class='form-group'>
    <label>Select city of Japan:</label>
    <div w-chz items='cities'>
      <div ng-show='item'>
        Code: <b>{{item.id}}</b>, Name: <b>{{item.label}}</b>
      </div>
      <div ng-hide='item'>
        No city selected
      </div>
    </div>
  </div>
</form>
angular.module('angular-w-demo')
.controller('ChzTemplatingsCtrl', function($scope) {
  $scope.cities = [
    {id: 'JP SPK', label: 'Sapporo'},
    {id: 'JP SDJ', label: 'Sendai'},
    {id: 'JP TYO', label: 'Tokyo'},
    {id: 'JP NGO', label: 'Nagoya'}
  ];
});
